# TechnicalWriter
Test tasks for candidates on position of technical writer
===========================================================================================

There are a total of three tasks that need to be completed within 24 hours.

 >>> The folder "Description" contains the task descriptions exported from Jira.

 >>> The folder "Result" includes the actual documents that need to be updated. The results should be committed and pushed to Git.

The complete documentation is available here as a reference: Raynet Scan Engine Documentation.
